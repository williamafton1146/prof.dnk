О платформе

ПрофДНК - это современная онлайн-платформа для профессиональной ориентации и психодиагностики.


Возможности платформы:

  • Конструктор диагностических методик
  • Прохождение тестов по уникальной ссылке без регистрации
  • Хранение результатов и метрик по каждому клиенту
  • Два вида отчетов: для клиента и для профориентолога
  • Экспорт отчетов в DOCX и HTML форматах

Пройти тест по ссылке

Вставьте ссылку на тест для начала прохождения.
Формат: https://profdnk.ru/?test=AbCdEfGh

Как это работает:

  1. Получите ссылку на тест от профориентолога
  2. Вставьте ссылку в поле выше
  3. Нажмите "Загрузить тест"
  4. Заполните ваши данные (имя, фамилия)
  5. Пройдите тестирование онлайн
  6. Получите результаты сразу после завершения
