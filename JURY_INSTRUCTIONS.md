## 🚀 Запуск проекта (3 команды)

### 1. Распаковать архив
```bash
tar -xzf profdnk-platform.tar.gz
cd profdnk-platform
```

### 2. Запустить Docker Compose
```bash
docker-compose up --build
```

Подождите 30-60 секунд пока все сервисы запустятся.

### 3. Открыть в браузере

- **Frontend**: http://localhost:3000
- **API Docs**: http://localhost:8000/docs
- **Backend**: http://localhost:8000

## 🎭 Демонстрация возможностей

### Сценарий 1: Вход в систему

1. Откройте http://localhost:3000
2. Нажмите "Войти"
3. Введите:
   - Username: `admin`
   - Password: `admin123`

### Сценарий 2: Создание опросника

1. После входа перейдите на http://localhost:3000/builder
2. Введите название: "Диагностика профессий"
3. Добавьте вопросы разных типов:
   - Текстовый вопрос: "Опишите вашу идеальную профессию"
   - Выбор: "Какой формат работы вам комфортен?"
   - Шкала 1-10: "Оцените коммуникабельность"
4. Нажмите "Сохранить и получить ссылку"
5. Ссылка скопируется автоматически!

### Сценарий 3: Прохождение теста клиентом

1. Откройте скопированную ссылку (или http://localhost:3000/test/1)
2. Введите имя: "Иван Тестовый"
3. Email (опционально): test@example.com
4. Нажмите "Начать тестирование"
5. Отвечайте на вопросы (видно анимации и прогресс-бар)
6. После завершения - **Confetti** 🎉
7. Скачайте отчет

### Сценарий 4: Просмотр отчетов

**Для клиента (DOCX):**
http://localhost:8000/reports/{session_token}/client

**Для психолога (HTML с AI-инсайтами):**
http://localhost:8000/reports/{session_token}/psychologist

## 🛠️ Если что-то пошло не так

### Порты заняты?
```bash
# Остановить
docker-compose down

# Изменить порты в docker-compose.yml
# 3000 → 3001, 8000 → 8001

# Запустить снова
docker-compose up --build
```

### База данных не работает?
```bash
# Пересоздать volumes
docker-compose down -v
docker-compose up --build
```

### Нужны демо-данные?
```bash
pip install requests
python seed_demo_data.py
```
