# 🚀 Быстрый старт ПрофДНК Platform

## За 3 минуты до работающего приложения!

### Шаг 1: Запуск через Docker

```bash
# Запустить все сервисы
docker-compose up --build
```

Дождитесь сообщений:
- ✅ `profdnk_db` - База данных готова
- ✅ `profdnk_backend` - API запущен на :8000
- ✅ `profdnk_frontend` - Frontend на :3000

### Шаг 2: Открыть приложение

Откройте браузер:
- **Frontend**: http://localhost:3000
- **API Docs**: http://localhost:8000/docs

### Шаг 3: Первый вход

**Администратор (создается автоматически):**
```
Username: admin
Password: admin123
```

### Шаг 4 (Опционально): Создать демо-данные

В новом терминале:

```bash
# Установить requests
pip install requests

# Запустить скрипт
python seed_demo_data.py
```

Получите готовый опросник с примерами!

## 🛠️ Устранение проблем

### Порты заняты?

Измените в `docker-compose.yml`:

```yaml
ports:
  - "3001:3000"  # Вместо 3000
  - "8001:8000"  # Вместо 8000
```

### База данных не инициализируется?

```bash
# Удалить volumes и пересоздать
docker-compose down -v
docker-compose up --build
```

### Frontend не собирается?

```bash
# Пересобрать только frontend
docker-compose up --build frontend
```
