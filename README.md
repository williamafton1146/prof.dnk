# Платформа ПрофДНК - Бэкенд

Этот репозиторий содержит бэкенд для платформы ПрофДНК — онлайн-конструктора психологических тестов с автоматической генерацией отчётов.

## Архитектура

- **Backend:** Python + FastAPI
- **ORM:** SQLAlchemy
- **База данных:** PostgreSQL
- **Аутентификация:** JWT
- **Отчёты:** DOCX (python-docx), HTML (Jinja2)
- **Деплой:** Docker Compose

## Запуск

1.  Убедитесь, что у вас установлен Docker и docker-compose.
2.  Клонируйте этот репозиторий.
3.  Перейдите в директорию `backend`.
4.  Выполните команду:
    ```bash
    docker-compose up --build
    ```
5.  Приложение запустится на `http://localhost:8000`.
6.  Swagger UI будет доступен по адресу `http://localhost:8000/docs`.

## Ключевые особенности

- Ролевая модель: Администратор, Психолог, Клиент.
- Управление доступом психологов.
- Конструктор тестов с поддержкой формул.
- Генерация отчётов в реальном времени.
- Безопасная аутентификация и хеширование паролей.
- Загрузка аватарок.
- Возможность клонирования тестов.

## Структура API

- `/auth/login` - Аутентификация.
- `/me/` - Профиль психолога.
- `/tests/` - CRUD тестов.
- `/public/t/{id}/...` - Публичные маршруты для прохождения теста.
