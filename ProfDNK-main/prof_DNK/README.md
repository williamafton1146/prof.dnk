# ПрофДНК – платформа для профориентологов

## Описание
Веб-платформа для создания психологических тестов, прохождения клиентами по уникальным ссылкам и автоматической генерации DOCX/HTML-отчетов.

## Технологии
- Backend: FastAPI, PostgreSQL, SQLAlchemy, python-docx
- Frontend: Next.js 14, Tailwind CSS, React Query
- Контейнеризация: Docker Compose

## Запуск
1. Клонировать репозиторий
2. Заполнить `.env` файлы (примеры в `backend/.env.example`)
3. Выполнить `docker-compose up --build`
4. Открыть http://localhost:3000

## Первый вход
- Создайте администратора через базу данных (например, вставьте запись в таблицу users с is_admin=true). Или можно добавить скрипт для создания admin.

## Структура
- `/backend` – REST API
- `/frontend` – клиентское приложение

## Документация API
После запуска бэкенда: http://localhost:8000/docs

## Возможности
- Создание тестов с 10+ типами вопросов
- Импорт/экспорт тестов в JSON
- Генерация отчетов (DOCX, HTML) для клиента и психолога
- Личный кабинет психолога с QR-визиткой
- Админ-панель для управления психологами
- Формулы для расчета метрик
